
# 烧结过程 CO 减排优化模型

> 河北省研究生数学建模竞赛参赛作品  
> 基于数据驱动的风箱温度预测、CO 浓度实时预测及负压优化调控

## 项目简介

本项目针对钢铁烧结过程 CO 排放控制问题，利用提供的工业时间序列数据，完成了三个核心任务：

1. **问题一**：建立基于风箱负压的风箱温度预测模型（每个风箱独立建模），实现精准预测。
2. **问题二**：考虑时滞效应，建立烧结大烟道外排 CO 浓度的实时预测模型，并针对工况突变采用分阶段建模策略。
3. **问题三**：基于 CO 预测模型，通过差分进化算法优化 18 个风箱负压，使 CO 排放浓度最低。

项目采用 **LightGBM** 作为核心回归算法，结合丰富的特征工程（长时滞、滑动窗口统计、相邻风箱耦合等），取得了良好的预测精度与优化效果。

## 文件结构

| 文件 | 说明 |
|------|------|
| `config.py` | 所有超参数、文件路径、模型参数统一配置 |
| `data_loader.py` | 数据读取、南北侧温度合并、缺失值处理 |
| `feature_engineering.py` | 通用特征构造函数（滞后、滑动统计、差分等） |
| `models.py` | 问题一（风箱温度）和问题二（CO 浓度）模型训练 |
| `optimization.py` | 问题三差分进化优化（单进程） |
| `visualization.py` | 所有绘图函数（散点图、曲线图、对比图等） |
| `logger.py` | 日志模块，同时输出到控制台和文件 |
| `main.py` | 主程序入口，依次执行三个问题并保存结果 |
| `requirements.txt` | Python 依赖包列表 |
| `README.md` | 项目说明文档 |

## 输出结果

- **控制台**：实时打印各模型训练进度、评估指标（MAE/RMSE/R²）、优化结果等。
- **日志文件**：`modeling_log.txt`，记录全部运行过程的关键数值。
- **图片窗口**：程序运行中会弹出所有图表（问题一散点/曲线、问题二预测对比、问题三优化对比等），可手动保存。
- **CSV 数据**：所有绘图数据自动保存在 `csv_output/` 目录下。
